# Fortuna Chung — Profil Administratif

Production-ready resume site for an administrative professional, built with Next.js 15 (App Router), TypeScript, Tailwind CSS and Framer Motion. Optimized for Vercel.

## Install

```bash
npm install
npm run dev
```

## Deploy

Push to a GitHub repo and import into Vercel — zero configuration required.

## Editing content

All resume data lives in a single file: `lib/data.ts`.
